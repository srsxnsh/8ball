## Install

simply run:
```
>curl -fsSL https://github.com/srsxnsh/8ball/blob/main/8ball.sh -o 8ball.sh
>chmod +x 8ball.sh
>sudo mv 8ball.sh /usr/local/bin/8ball
```
## Commands

- shake     : Shakes the 8ball
- --help    : Displays help menu
- --version : Displays current version
